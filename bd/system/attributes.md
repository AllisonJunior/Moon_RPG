# Atributos

Diferentemente do sistema D&D 5e padrão, o Sistema Moon Trabalha só com 5 atributos:

- Força 
- Destreza
- Constituição
- Inteligência
- Carísma

Todos esses atributos possuem valor base igual a 8, e você jogador tem acesso a um total de 28 pontos de atributos para distribuir a seu bel prazer, seguindo a simples regra de que nenhum atributo pode exceder o límite de 18. 

Note que o personagem ganha pontos de atributos conforme ele sobe de nível, que não estão restringidos a regra de atributo com valor máximo igual a 18.
